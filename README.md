# Mazaa Eatery Media Repository

This repo stores public media assets (images, icons, banners, PDFs) used by Mazaa Eatery apps/websites.

## Folder structure
- branding/         logos, icons, badges
- home/             homepage banners, promos, section graphics
- menu/             dish images (breakfast, thalis, ingredients)
- ui/               icons/illustrations used in the UI
- legal/            policy PDFs
- _meta/            asset index + upload guidelines

## Naming rules
- Use kebab-case (e.g., slide-01-hero-banner.jpg)
- No spaces in filenames
- Keep filenames stable once used in production

## Suggested usage
Enable GitHub Pages for stable public URLs.
