# Upload Checklist (Mazaa Media)

Before uploading:
-  Rename in kebab-case (no spaces)
-  Optimize size (images < 500KB when possible)
-  Use JPG/WebP for photos, PNG for logos/badges
-  Place in correct folder (branding/home/menu/etc.)

After uploading:
-  Update _meta/assets-index.json if needed
-  Verify the file opens from GitHub Pages / direct URL
